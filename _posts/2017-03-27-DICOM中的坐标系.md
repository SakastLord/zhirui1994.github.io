# DICOM中的坐标系

## Patient Coordinate System 患者坐标系统
分类： BIPED（双足动物）和 QUADRUPED （四足动物）

以人为例，定义患者坐标：
- X轴，正方向为从右到左
- Y轴，正方向从前胸到后背
- Z轴，正方向为脚底到头顶

# RCS( Reference Coordinate System) 参考坐标系统
RCS坐标系用于度量DICOM参考帧。
RCS是右手“笛卡尔坐标系统”，笛卡尔坐标系是我们常见的三维空间坐标系统，根据x、y、z三轴的相互顺序分为右手坐标系和左手坐标系
